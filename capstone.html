Andrew's Capstone Project

The Oregon Gardening Application

The Oregon Gardening application will be created by a three member team to benfit Oregon State University's Department of Horticulture. Our team is in the process of will be creating a mobile application for iOS and Android that will allow users to access frost date information for any location in Oregon, based on NOAAâ€™s most recent climate data. We will be using the Ionic Framework, which enables mobile app development using common web technologies. We are excited to work with out client to produce an application that will be beneficial to farmers and gardening enthusiasts in Oregon and potentially beyond.
